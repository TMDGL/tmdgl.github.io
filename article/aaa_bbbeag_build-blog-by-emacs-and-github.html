通过emacs ox-publish与github page结合建立简便、极少依赖、高度可定制的 的静态博客。

1. Emacs使用ox-publish

1.1. 操作平台

  • Window Home 10
  • GNU Emacs 27.1 (build 1, x86_64-w64-mingw32) of 2020-08-11
  • Org mode version 9.3 (Emacs build in)

1.2. Emacs 配置

参考教程：Publishing Org-mode files to HTML 

;; plug request(but emacs default run ox-publish)
(require 'ox-publish)

;; publish configure
(setq org-publish-project-alist
      '(

	;; org notes export to html
	("org-notes" 	                
	 :base-directory "~/blog/org"
	 :base-extension "org"
	 :publishing-directory "~/blog/"
	 :recursive t
	 :publishing-function org-html-publish-to-html
	 :auto-preamble t
	 )

	;; org static file
	("org-static"                   
	 :base-directory "~/blog/org"
	 :base-extension "css\\|js\\|png\\|jpg\\|gif\\|ogg\\|swf"
	 :publishing-directory "~/blog"
	 :recursive t
	 :publishing-function org-publish-attachment
	 )

	("org-blog" :components ("org-notes" "org-static"))))

1.4. HTML转换

在org源码目录下（本文为 ~/blog/org ）建立一个index.org文件。任意输入 一些测试内容然后运行 

M-x org-publish-project [Enter] org-blog [Enter]

中文可能会在minibuffer出现提示 

Select system code(default utf-8)

直接回车选中utf-8.

打开emacs publish的文件夹（本文为 ~/blog ）打开index.html文件即可。

2. 博客css美化

2.1. css渲染

org文件中添加 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="_your css path_">

此博客css文件为org官网的css渲染文件worg.css

2.1.1. css寻找方式

笔者是在Publishing Org-mode files to HTML中直接下载页面整个文件，可以 得到一个html文件和文件夹，打开文件夹比重就包含worg.css文件。此文件为主 要的渲染文件。其他的css文件作用不明，js文件应该是提供html数学公式显示。

也可在其他的相似的org构建的博客去寻找，参考资料中给出了几个可以参考的 链接。此外，Org官网也有很多博客可以参考。

如果你不嫌麻烦，也可手撸css文件。可在 The Org Manual 查看至于html export出的对象和属性(应该是叫这个，html不了解)。

3. Org

3.1. Template模板

Org中的模板后缀名不一定必须为org，可以避免ox-publish将模板文件导出。其 余部分Publishing Org-mode files to HTML中存在详细介绍。

3.2. 图片/表格添加描述

参考 Org Manual 12.7 Images 此处可用file协议直接显示图，添加了描述。 

#+CAPTION: This is the caption for the next figure link (or table)
#+NAME:   fig:SED-HR4049
[[./img/a.jpg]] 

# Just for show, No picture here.

NOTE :在emacs中预览图片需开启iimage-mode。

5. 注意事项

  • 文件引用一定要用相对路径。否则部署到github会会访问失败。
  • "_" 会导致下标显示，解决方法 1
  • 插入图片直接快捷键C-c C-l的情况下如果不指明具体协议。默认为file文件 协议。此时导出为html时不会直接显示图片。
  • Github静态资源似乎国内不能直接访问，图片不能加载。

  • 文件保存必须为utf-8格式

    配置文件中加入 

    (setq-default buffer-file-coding-system 'utf-8)
